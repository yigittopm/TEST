
# Go Docker Hub

A simple API and automatic Docker Hub image refresh with Go

## Installation

Pull this repo and follow the steps below

```bash
  git clone https://github.com/WeLedger/wl-auth.git
  cd wl-auth
```

```bash
  make up
```
